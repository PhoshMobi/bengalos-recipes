# Qualcomm SDM670 based phones

Supported devices

- Google Pixel 3A (google,sarg)

### Flashing U-Boot

First we need to flash U-Boot to the slot you want to use. You can fetch a prebuilt
u-boot from the [tauchgang project][tauchgang].

```sh
sudo apt install fastboot
fastboot flash boot_a u-boot-google-sargo.img
```

### Flash the BengalOS Image

After flashing U-Boot to the board we can build and flash the image:

```sh
make bengalos-sdm67-immutable
fastboot flash userdata build-sdm670-immutable/BengalOS_<version>.raw
```
